# ifconfig

ifconfig用于查看和配置Linux系统的网络接口

查看所有网络接口及其状态

```text
$ ifconfig -a
```

使用up和down命令启动或停止某个接口

```text
$ ifconfig eth0 up  
$ ifconfig eth0 down

$ifup eth0
$ifdown eth0
```

更多示例：[Ifconfig: 7 Examples To Configure Network Interface](http://www.thegeekstuff.com/2009/03/ifconfig-7-examples-to-configure-network-interface/)
