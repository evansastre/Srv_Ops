# finger

```text
-l Lists the user's account name, real name, user-specific directory, shell used for login, login time, transfer address, email status, and plan file and program file content.
-m Excludes the real name of the found user.
-s Lists the user's account name, real name, login terminal, idle time, login time, and address and phone number.
-p Lists the user's account name, real name, user-specific directory, shell used for login, login time, transfer address, email status, but does not display the user's plan file and program file content.
```
