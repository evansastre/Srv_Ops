# wget

使用wget从网上下载软件、音乐、视频

```text
$ wget http://prdownloads.sourceforge.net/sourceforge/nagios/nagios-3.2.1.tar.gz
```

下载文件并以指定的文件名保存文件

```text
$ wget -O taglist.zip http://www.vim.org/scripts/download_script.php?src_id=7701
```
